### Starting the Project
1. Create a .env file in the root directory.
2. Insert
   ```
   PORT=5000 or any other of your choice
   MONGO_URI=Your_MongoDb_uri
   ```
3. Run ``` npm install ```
4. Run ```npm run dev```
5. Your Server will be up and running
